# Introduction to Okta Identity Governance

This section provides an introduction of Okta Identity Governance (OIG),
what it is, what its architecture looks like, key data objects,
integrations and other uses of the OIG components.

This section, and the rest of the document, will focus on the access
governance capabilities (entitlement management, access requests, access
certification and governance reporting). It assumes the reader is
familiar with the other SKUs in the OIG bundle - Okta Lifecycle
Management and Okta Workflows.
