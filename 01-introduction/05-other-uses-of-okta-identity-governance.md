## Other Uses of Okta Identity Governance


Other Okta products or features leverage OIG capabilities.

Okta Privileged Access leverages the Access Request capability for
just-in-time approval of privileged access, and is due to release an
access certification function for managed service accounts.

The Okta [<u>Govern Admin
Roles</u>](https://help.okta.com/oie/en-us/content/topics/security/governance-admin-roles/govern-admin-roles.htm)
feature leverages both access requests and access certification.
